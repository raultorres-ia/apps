# Descomposició i Sumes – 3r Primària

Aplicació web senzilla per practicar la descomposició de nombres i les sumes (pensada per a 3r de Primària). Tot el contingut és en català.

## Ús ràpid (Windows)

- Obre el fitxer `index.html` fent doble clic. No cal instal·lar res.
- Pestanyes disponibles:
  - "Descomposició": descompon nombres en centenes, desenes i unitats.
  - "Sumes": resol sumes verticals amb 2 o 3 xifres.

## Idees didàctiques

- Demana a l'alumnat que expliqui l'estratègia (centenes, desenes, unitats) abans de prémer "Comprova".
- Usa "Pista" per fer visible el valor posicional o el primer pas de la suma.
- Treballa en gran grup projectant la pantalla i demanant justificacions.

## Personalització

- Dificultat de descomposició: 0–99, 100–499 o 500–999.

## Crèdits

Creat per a activitats de 3r de Primària.
