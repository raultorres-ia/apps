# Dictat matemàtic de figures planes

Aquesta aplicació web interactiva està dissenyada per ajudar els nens i nenes de tercer de primària a practicar dictats matemàtics de figures planes. L'aplicació reprodueix àudios amb instruccions pas a pas que els usuaris han de seguir per dibuixar figures a la seva llibreta.

## Millores visuals per a nens de primària

S'han realitzat les següents millores visuals per fer l'aplicació més atractiva i amigable per a nens de tercer de primària:

*   **Tipografia amigable:** S'ha incorporat la font "Fredoka" per donar un aspecte més juvenil i llegible.
*   **Colors vibrants:** La paleta de colors s'ha ajustat per ser més alegre i captar l'atenció dels infants.
*   **Ombres suaus i efectes interactius:** S'han suavitzat les ombres dels elements i s'han afegit subtils efectes de "hover" a les targetes d'àudio per millorar la interactivitat.
*   **Botons i icones més grans:** Els elements interactius com els botons i les icones s'han fet lleugerament més grans per facilitar la seva interacció.
*   **Espaiat millorat:** S'ha ajustat l'espaiat entre elements per a una interfície més neta i menys carregada.

## Com utilitzar l'aplicació

1.  **Obrir `index.html`:** Simplement obre el fitxer `index.html` en el teu navegador web preferit.
2.  **Escoltar els àudios:** Fes clic als botons "Reproduir" de cada targeta d'àudio per escoltar les instruccions.
3.  **Dibuixar:** Segueix les instruccions de l'àudio per dibuixar les figures a la teva llibreta.
4.  **Controlar la reproducció:** Utilitza els botons "Pausa" i "Tornar a començar" segons sigui necessari.

## Estructura del projecte

*   `index.html`: La pàgina principal de l'aplicació.
*   `style.css`: Els estils visuals de l'aplicació.
*   `script.js`: La lògica interactiva per a la reproducció d'àudios.
*   `assets/audio/`: Conté els fitxers d'àudio (`.mp3`) utilitzats en els dictats.
